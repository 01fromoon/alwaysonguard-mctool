## 🇬🇧 ENGLISH

### ⚡ Overview

AlwaysOnGuard is a modern, multi-tool terminal app for Minecraft admins and enthusiasts.  
It features server info, DNS & IP tools, port scanner, password guesser, bot stress test, a fully customizable **FakeProxy honeypot** and more – all with multi-language support!  

---

### 🚀 Features

- 🟩 **Server Info**: Query any Minecraft server (`server`)
- 🌐 **DNS & IP Tools**: Lookup DNS records, IP info/history (`dns`, `ipinfo`, `iphistory`)
- 🔍 **Port Scanner**: Scan single/multiple IPs, ranges, wildcards, files
- 🤖 **BotsAttack**: Stress test with fake bots (for your own server!)
- 🛡️ **FakeProxy**: Honeypot to fool attackers & log connections, fully customizable
- 🔐 **Password Guesser**: Guess weak passwords for any nickname
- 🔤 **Multi-language**: 🇬🇧 English, 🇹🇷 Türkçe, 🇪🇸 Español

---

### 🛠️ Installation

#### 1. Clone the Repo

```bash
git clone https://github.com/youruser/alwaysonguard.git
cd alwaysonguard
```

#### 2. Install Requirements

```bash
pip install -r requirements.txt
```
- Python 3.7+ required

#### 3. Run the Program

```bash
python main.py
```

#### 4. Choose Your Language, Enjoy!

---

### 💻 Usage

Type `help` to see available commands.

Example commands:
```bash
server play.alwaysonguard.net
dns alwaysonguard.net
ipinfo 1.1.1.1
iphistory mojang.com
scan 192.168.1.1 1-1000
botsattack 127.0.0.1 25565 20
password 01fromoon
fakeproxy --port 25566 --motd "Honeypot!" --version "1.20.1"
```

---

### 📝 Configuration

- **Languages:** Change anytime with `language tr`, `language en`, `language es`
- **FakeProxy:** Fully interactive or command-line options
- **Logs:** All logs saved under `logs/`  
- **Config files:** See `locale/messages.json` for localizations
